# (C# 코딩)
## 개요
	-C# 프로그래밍학습
	-1줄소개: 사용자키보드입력을받아서처리하는프로그램
	-사용한플랫폼: -C#, .NET Windows Forms, Visual Studio, GitHub
	-사용한컨트롤:-Label, TextBox, ListBox, Button, ListView, MenuStrip, ToolStripMenuItem
	-사용한기술과구현한기능:
		-Visual Studio를이용하여UI 디자인
		-Listview컨트롤을이용하여파일비교시각화
